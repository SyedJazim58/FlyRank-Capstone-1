# FlyRank AI Internship — Welcome Page

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
![Status](https://img.shields.io/badge/Status-Development-yellow)

A polished welcome interface for the FlyRank AI Internship program, built with plain HTML and CSS. This is the capstone project for the Frontend AI Intern track, demonstrating AI-assisted development workflows.

## Preview

Open `index.html` in any browser to see the page — no build step needed.

The page features a full-screen gradient hero, feature cards, a values section, and a responsive layout that adapts from desktop to mobile.

## Features

- **Hero section** — Dark gradient landing with animated scroll indicator
- **About cards** — Four feature highlights (Projects, Mentors, Learning, Career)
- **Values section** — Numbered program values with descriptions
- **Responsive design** — Mobile-first layout with breakpoints at 768px and 480px
- **Smooth interactions** — Hover animations on cards, buttons, and nav links
- **Accessible markup** — Semantic HTML, ARIA attributes, proper heading hierarchy

## Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, gradients, animations, responsive grid
- **Inter** — Google Font for clean typography
